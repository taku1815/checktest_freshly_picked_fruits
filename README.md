# もぎたて

## 環境構築
1. docker-compose exec php bash
2. composer install
3. .env.exampleファイルから.envを作成し、環境変数を変更
4. php artisan migrate
5. php artisan db:seed
   
## 使用技術(実行環境)
- PHP 8.44
- Laravel 8.6.12
- MySQL 8.0
